![image](https://www.dropbox.com/s/9b7snej72etq1v5/vivaldi_ZQC6mZ8345.gif?raw=1)
## 🚀Features:

- 🧩 Add tasks
- ✅ toggle tasks as (un)completed
- 🗑️ delete tasks
- 🪨 persist data between reloads
- 📁 filter by only completed tasks
- 📂 filter by only uncompleted tasks
